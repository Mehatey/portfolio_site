01 Concept

What feels beautiful?
What feels broken?

B + b is a public interaction project asking one question: what feels beautiful and what feels broken. It collected 285+ responses and 15+ street interviews across NYC.

The project created a space where strangers could see themselves in each other, finding shared language for experiences that usually stay private.

Street interview, NYC public interaction
Vercel · Backend · Web Interface
02 Process

One question, many voices.

The web interface allowed anyone to submit responses anonymously. Street interviews across NYC added a face-to-face dimension, capturing spontaneous reactions and deeper reflections.

All responses fed into a shared space where patterns emerged, beauty and brokenness often pointed to the same things: relationships, cities, routines, time.
